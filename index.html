My Projects

The Parliament Connect

Python | JavaScript | Flask | Node.js | React.js | Elasticsearch | PostgreSQL | sentence-transformers | AWS

EPACS - Effective Parliament Archive Connect System, a fully featured semantic search engine, tailor made for the Parliament of India as a part of Smart India Hackathon 2022 grand finale. The prototype aims to provide a comprehensive search engine for the Parliament records and a dashboard to manage the data pipeline. The project was initially intended to help eliminate the redundant questions asked in the Parliament. Later the project evolved to function as the primary search engine to retrieve the Parliament records.

MarkAnn

Python | Generative AI | Async-Programming | WebSocket | FastAPI | Qdrant | AWS | GitHub Actions(CD)

A Telegram bot that provides real-time press releases from the companies listed on Bombay Stock Exchange (BSE). The bot summarizes the press release documents and provides the user with the summarized content along with the type of the press release. The project is released as open-source under GPL-2.0 license and is currently at the alpha stage of development.

The Ayush Connect *

Python | Generative AI | Vision Transformers | FastAPI | PostgreSQL | Qdrant | Flutter

An AI powered platform that provides information about Ayurvedic medicinal plants for the Ayurvedic pharmaceutical industry. The platform uses a custom vision transformer model to identify the medicinal plants with high accuracy. At the heart of the platform is a chatbot - PlantGPT, that helps the users to interactively get information about the medicinal plants.

Chequer

Python | YoloV9 | Amazon Textract | FastAPI | PostgreSQL | Async-Programming

Chequer is a modern and efficient solution for automating the cheque clearing process. It provides a streamlined workflow for processing cheques, reducing manual effort and improving accuracy. The system is built with a custom YOLOv9 model to detect the cheque details and Amazon Textract query service to extract the text from the cheque using natual language queries. At the heart of the system is an asynchronous queue system that provides a scalable and efficient way to process the cheques.

RL Portfolio Allocator

Python | Reinforcement Learning | Streamlit

A portfolio allocation system that uses Reinforcement Learning to optimize the portfolio allocation strategy. The system uses multiple RL agents to learn the optimal allocation strategy for the given market conditions.

* The project is currently under development and is expected to be released in the upcoming months.
